27 มิ.ย. 2026 · อ่าน ~8 นาที

เราอ่าน งบกำไรขาดทุน (ทำเงินได้แค่ไหน) และ งบกระแสเงินสด (กำไรกลายเป็นเงินสดจริงไหม) มาแล้ว ตอนปิดซีรีส์นี้เราจะอ่านงบชุดสุดท้าย — งบดุล — แล้วทำสิ่งที่สำคัญที่สุด: เอางบทั้ง 3 มาต่อกันให้เห็นว่ามันคือเรื่องเดียวที่เล่าจาก 3 มุม

งบดุลคือ "ภาพถ่าย ณ จุดเวลาหนึ่ง"

ต่างจากงบกำไรขาดทุนและงบกระแสเงินสดที่เล่า "ช่วงเวลา" (ทั้งปี) งบดุลคือ ภาพถ่าย ณ วันสิ้นงวด — บอกว่า ณ วินาทีนั้นบริษัทมีอะไร เป็นหนี้ใคร และเหลือเป็นของเจ้าของเท่าไหร่

สินทรัพย์ = หนี้สิน + ส่วนของเจ้าของ
(ของที่มี = เงินที่ยืมคนอื่นมา + เงินของเราเอง)

สมการนี้ต้อง "บาลานซ์" เสมอ (เลยเรียกงบดุล / Balance Sheet) — เพราะทุกอย่างที่บริษัทมี ต้องมาจากเงินสองแหล่งเท่านั้น: ยืมเขามา หรือเป็นของเจ้าของเอง

งบดุลร้านกาแฟ ณ สิ้นปี

ต่อจากปีที่ร้านขยายไปขายส่ง (จากตอนที่ 2) งบดุลสิ้นปีหน้าตาประมาณนี้:

สินทรัพย์ (Assets)บาท
เงินสด164,000
ลูกหนี้การค้า (ขายเชื่อ ยังไม่เก็บเงิน)180,000
สินค้าคงคลัง90,000
อุปกรณ์ร้าน (สุทธิหลังค่าเสื่อม)500,000
รวมสินทรัพย์934,000
หนี้สิน + ส่วนของเจ้าของบาท
เงินกู้ธนาคาร (หนี้สิน)300,000
ทุนจดทะเบียน (เงินเจ้าของลงตอนแรก)450,000
กำไรสะสม (กำไรที่เก็บไว้ในกิจการ)184,000
รวมหนี้สิน + ส่วนของเจ้าของ934,000

สังเกตว่าสองฝั่ง เท่ากันที่ 934,000 — บาลานซ์พอดี

ดูอะไรในงบดุล

  • หนี้เทียบความสามารถจ่าย — เงินกู้ 300,000 เทียบกำไรจากการดำเนินงาน 250,000/ปี (จากตอน 1) ≈ 1.2 เท่า → จัดการไหว ไม่ใช่หนี้ท่วม
  • ลูกหนี้ + สต็อก พองไหม — รวมกัน 270,000 จมอยู่ใน working capital นี่แหละคือตัวที่ทำให้กำไร 184,000 กลายเป็นเงินสดติดลบในตอนที่ 2
  • สภาพคล่อง — สินทรัพย์หมุนเวียน (เงินสด+ลูกหนี้+สต็อก = 434,000) ควรพอจ่ายหนี้ระยะสั้น ถ้าน้อยกว่าหนี้สินหมุนเวียนคือสัญญาณตึง
  • Goodwill — ถ้าบริษัทไปซื้อกิจการอื่นมาแพง ส่วนเกินจะกลายเป็น "Goodwill" ก้อนใหญ่ในสินทรัพย์ ซึ่งเสี่ยงถูกตัดมูลค่าทิ้ง (impairment) ถ้าดีลไม่เวิร์ก — ต้องระวังเป็นพิเศษ

หัวใจของซีรีส์: เชื่อม 3 งบเข้าด้วยกัน

นักลงทุนมือใหม่มักอ่านงบ 3 ชุดแยกกัน แล้วโดนหลอก ความจริงคือมันต่อกันสนิท — ลองดูจากร้านกาแฟเรา:

เส้นเชื่อมเกิดอะไรขึ้น
กำไร → กำไรสะสม กำไรสุทธิ 184,000 จากงบกำไรขาดทุน ไหลเข้าไปเป็น "กำไรสะสม" 184,000 ในส่วนของเจ้าของในงบดุล (ส่วนที่ไม่ได้ปันผลออก)
งบดุล → งบกระแสเงินสด ลูกหนี้ +180,000 และสต็อก +90,000 ที่โผล่ในงบดุล คือตัวเดียวกับที่ถูกหักในงบกระแสเงินสด ทำให้ CFO ติดลบ (เงินจมใน working capital)
เงินสดปลายงวด → งบดุล เงินสดต้นปี 200,000 + CFO (−36,000) = เงินสดปลายงวด 164,000 ซึ่งตรงเป๊ะกับช่อง "เงินสด" ในงบดุล
งบกำไรขาดทุนบอก "ทำเงินได้แค่ไหน" · งบกระแสเงินสดบอก "เป็นเงินสดจริงไหม" · งบดุลบอก "สะสมเป็นฐานะอะไร" — สามมุมของธุรกิจเดียวกัน ถ้าอ่านครบและเชื่อมได้ ตัวเลขจะโกหกคุณยากขึ้นมาก

สรุป: 3 คำถามที่ต้องตอบจากงบดุล

  1. หนี้อยู่ในระดับที่กำไรของกิจการรับไหวไหม (ไม่ใช่ธุรกิจที่ต้องกู้มาประคอง)?
  2. ลูกหนี้/สต็อก/Goodwill มีก้อนไหนพองผิดปกติจนเสี่ยงไหม?
  3. ตัวเลขในงบดุลเชื่อมกับอีก 2 งบได้สนิทไหม (กำไรสะสม, เงินสด, working capital)?

Kill Conditions — งบดุลแบบไหนที่ผมจะถอย

  • หนี้พุ่งเร็วกว่ากำไรต่อเนื่อง (Net Debt/EBIT สูงขึ้นเรื่อยๆ) โดยไม่มีเหตุผลการลงทุนที่ให้ผลตอบแทนชัด
  • ลูกหนี้หรือสต็อกโตเร็วกว่ายอดขายหลายปีติด (เงินจม คุณภาพกำไรต่ำ)
  • Goodwill ก้อนใหญ่จากดีลแพง + สัญญาณว่าดีลไม่เวิร์ก = เสี่ยง impairment
  • ส่วนของเจ้าของถูกลดทอนเรื่อยๆ (จำนวนหุ้นเพิ่มจาก dilution) หรือ equity ติดลบ

จบซีรีส์ — แล้วยังไงต่อ

ถึงตรงนี้คุณมีเครื่องมืออ่านงบครบทั้ง 3 ชุด และรู้วิธีเชื่อมมันเข้าด้วยกันแล้ว นี่คือพื้นฐานที่นักลงทุนสาย Fundamental ใช้ทุกวัน ก้าวต่อไปคือเอาไปใช้กับ บริษัทจริง — อ่านงบจริง หา moat จริง ตั้ง Kill Conditions จริง ซึ่งจะเป็นเนื้อหาชุดถัดไปของเว็บนี้
