เป็นร้อยปีที่จรวดถูกใช้ครั้งเดียวแล้วทิ้ง — เหมือนเผาเครื่องบินทั้งลำหลังบินจบเที่ยวเดียว SpaceX ทำลายกฎข้อนั้นด้วยการ เอา booster กลับมาใช้ซ้ำ และนั่นทำให้ต้นทุนการขึ้นอวกาศ ดิ่งลงจนเปิดธุรกิจใหม่ทั้งหมด (Starlink) บทนี้อธิบายว่า SpaceX หาเงินยังไง คูเมืองอยู่ตรงไหน และความเสี่ยงคืออะไร

SpaceX เป็นบริษัทเอกชน ไม่ได้จดทะเบียนในตลาดหุ้น — ไม่มีงบการเงินสาธารณะให้ตรวจสอบ ตัวเลข/มูลค่าที่อ้างถึงมาจากข้อมูลสาธารณะและรอบระดมทุนเอกชน (เป็นการประมาณ) บทนี้เน้นเข้าใจ โมเดลธุรกิจและคูเมือง เพื่อการศึกษา ไม่ใช่คำแนะนำซื้อขาย และไม่ใช่หลักทรัพย์ที่ซื้อขายได้ทั่วไป

1. ธุรกิจนี้หาเงินจากอะไร

แก่นของ SpaceX คือ บริการส่งของขึ้นอวกาศ (launch services) — ลูกค้าจ่ายเงินให้ SpaceX พาดาวเทียม, นักบินอวกาศ หรือสัมภาระขึ้นสู่วงโคจร ลูกค้าหลักมี 2 กลุ่ม: ภาครัฐ (NASA, กองทัพสหรัฐ) และ เอกชน (บริษัทดาวเทียมสื่อสาร/สำรวจ)

แต่สิ่งที่ทำให้ SpaceX ต่างจากทุกเจ้าในประวัติศาสตร์คือ การใช้จรวดซ้ำ:

จรวดเดิมทิ้ง booster (ท่อนล่างที่แพงและซับซ้อนที่สุด) ลงทะเลทุกเที่ยว SpaceX ให้มันบินกลับมาลงจอด — ล่าสุดถึงขั้นให้หอ "Mechazilla" ใช้แขนหนีบจับกลางอากาศ — แล้วเอาไปเติมเชื้อเพลิงบินใหม่ เหมือนสายการบินที่ใช้เครื่องบินลำเดิมซ้ำ ไม่ใช่ทิ้งทุกเที่ยว

2. คูเมือง: ต้นทุนจากการใช้ซ้ำ

คูเมืองของ SpaceX ไม่ใช่ "จรวดแรงที่สุด" แต่คือ ต้นทุนต่อกิโลกรัมที่ถูกที่สุดในโลก อย่างมีนัยสำคัญ ซึ่งมาจากการใช้ซ้ำ + สเกล + vertical integration (ผลิตเองเกือบทุกชิ้น):

นี่คือ Scale Economies + Process Power แบบ Helmer — ยิ่งปล่อยบ่อย ยิ่งเรียนรู้ ยิ่งใช้ซ้ำได้ดี ต้นทุนยิ่งลด คู่แข่งที่ยังทิ้งจรวดทุกเที่ยวจึงสู้ราคาแทบไม่ได้ และต้องใช้เวลาหลายปี กว่าจะไล่ตามเทคโนโลยีการลงจอดทัน

3. ขาที่สอง: Starlink

เมื่อต้นทุนการปล่อยถูกลงมาก SpaceX ทำสิ่งที่ฉลาด — ใช้จรวดราคาถูกของตัวเองสร้างธุรกิจใหม่ คือ Starlink เครือข่ายดาวเทียมอินเทอร์เน็ตวงโคจรต่ำ:

Starlink เปลี่ยน SpaceX จาก "บริษัทรับจ้างปล่อยจรวด" (รายได้เป็นโปรเจกต์) ไปสู่ธุรกิจที่มี รายได้ประจำแบบ subscription จากผู้ใช้ทั่วโลก — เงินก้อนนี้หมุนกลับมา fund การพัฒนา Starship และทำให้ flywheel หมุนเร็วขึ้น

4. สเกล: ทิ้งห่างทั้งโลก

ตัวเลขที่สะท้อนคูเมืองชัดที่สุดคือ cadence — ความถี่ในการปล่อยจรวด:

การที่ SpaceX ส่งมวลของขึ้นวงโคจรต่อปีมากกว่าทุกประเทศและบริษัทอื่นรวมกัน ไม่ใช่แค่สถิติสวยๆ — มันคือ กลไกที่ทำให้คูเมืองกว้างขึ้นเรื่อยๆ เพราะยิ่งปล่อยเยอะ ต้นทุนต่อเที่ยวยิ่งถูก และ Starlink ยิ่งขยายเร็ว

5. Flywheel: ยิ่งหมุนยิ่งถูก

วงล้อนี้คือหัวใจ — และเป้าหมายปลายทางคือ Starship จรวดที่ใหญ่และตั้งใจให้ ใช้ซ้ำได้ทั้งลำ (ไม่ใช่แค่ booster) ถ้าสำเร็จจะกดต้นทุนต่อ kg ลงอีกขั้น และเปิดทาง ขนคน/ของปริมาณมหาศาล — รวมถึงเป้าหมายระยะยาวเรื่องดาวอังคารที่เป็น optionality สุดขอบ

6. ตัวเลขและมูลค่า — ระวังตรงนี้

  • เป็นบริษัทเอกชน — ไม่มีงบการเงินสาธารณะ ตัวเลขรายได้/กำไรที่เห็นในข่าวมักเป็น การประมาณหรือหลุดบางส่วน ตรวจสอบยาก
  • มูลค่าอ้างอิงจากรอบระดมทุนเอกชน — เป็น valuation ของ private round ไม่ใช่ราคาตลาดที่ซื้อขายได้จริงทุกวัน
  • สภาพคล่องต่ำมาก — คนทั่วไปซื้อขายหุ้นโดยตรงแทบไม่ได้ ส่วนใหญ่เข้าถึงผ่าน กองทุน/SPV ทางอ้อม ที่มีค่าธรรมเนียมและความไม่โปร่งใสของตัวเอง
เพราะข้อมูลตรวจสอบยาก บทนี้จึงไม่ลงตัวเลขเป๊ะหรือมูลค่าเป้า — โฟกัสที่ "เข้าใจว่าทำไมมันเป็นธุรกิจที่ดี และความเสี่ยงอยู่ตรงไหน" ซึ่งสำคัญกว่าตัวเลขที่เชื่อถือไม่ได้

7. Bull Case

  • คูเมืองต้นทุนจากการใช้ซ้ำ + สเกล — นำห่างคู่แข่งหลายปี ยิ่งปล่อยยิ่งถูก
  • Starlink เป็นเครื่องจักรรายได้ประจำ — ตลาดอินเทอร์เน็ตทั่วโลก (รวมพื้นที่ห่างไกล/ทะเล/ การบิน) ใหญ่มากและ SpaceX แทบผูกขาด LEO broadband
  • Starship = option value มหาศาล — ถ้าใช้ซ้ำทั้งลำได้จริง เศรษฐศาสตร์อวกาศเปลี่ยนอีกขั้น

8. Bear Case / ความเสี่ยง

  • Starship ยังต้องพิสูจน์ — เป็นเดิมพันใหญ่ที่ใช้เงินมหาศาล ถ้า timeline เลื่อนยาว หรือไม่ถึงเป้า reuse ทั้งลำ option value จะหด
  • คู่แข่งและภูมิรัฐศาสตร์ — Blue Origin, Rocket Lab และโครงการรัฐของจีนกำลังพัฒนา reusable; การพึ่งสัญญารัฐ (NASA/กองทัพ) ทำให้การเมืองมีผล
  • Key-man และ governance — ผูกกับ Elon Musk สูง และเป็นเอกชนที่ไม่เปิดงบ/ตรวจสอบยาก

9. Kill Conditions — อะไรจะบอกว่าคูเมืองเริ่มพัง

ถึงจะลงทุนตรงไม่ได้ การมี Kill Condition ไว้ก็ช่วยให้ติดตามอย่างมีวินัย:

  • คู่แข่งทำ reusable booster ได้จริงในระดับ cadence ใกล้เคียง (reuse parity) → คูเมืองต้นทุนแคบลง ความได้เปรียบหลักเริ่มหาย
  • Starship ล้มเหลวซ้ำหลายปีจน timeline เลื่อนใหญ่ → option value หดและเงินทุนจม ในเดิมพันที่ยังไม่คืน
  • Starlink subscriber/ARPU โตช้าลงชัด หรือถูกแข่งด้านราคา → ขารายได้ประจำที่ fund Starship อ่อนลง flywheel หมุนช้า

สรุป

SpaceX เป็นกรณีศึกษาของธุรกิจที่ moat มาจาก ต้นทุนที่ถูกลงด้วยการใช้ซ้ำ + สเกล + vertical integration แล้วต่อยอดเป็นเครื่องจักรรายได้ประจำ (Starlink) ที่ fund เดิมพันถัดไป (Starship) จุดเปราะคือ Starship ที่ยังต้องพิสูจน์ และข้อจำกัดของการเป็น เอกชนที่ตรวจสอบยาก และซื้อขายไม่ได้ทั่วไป — น่าศึกษาในฐานะโมเดลธุรกิจชั้นเยี่ยม มากกว่าจะเป็นหุ้นที่ถือได้สะดวก
