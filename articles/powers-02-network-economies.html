อำนาจแรก (Scale Economies) ปกป้องฝั่งต้นทุน — แต่มีสนามที่โหดกว่านั้น: สนามที่คุณค่าของสินค้าเพิ่มขึ้นตามจำนวนคนที่ใช้มัน โทรศัพท์เครื่องเดียวในโลกไร้ค่า ตลาดที่ไม่มีผู้ซื้อไม่มีร้านมาเปิด — ในสนามแบบนี้ผู้ชนะไม่ใช่คนที่ถูกที่สุดหรือเก่งที่สุด แต่เป็นคนที่มาก่อนแล้วดึงคนได้ถึงมวลวิกฤต ตอนนี้เราจะไปดูอำนาจนี้ในที่ที่มันดุที่สุดในซีกโลกใต้: MercadoLibre ตลาดออนไลน์ที่ใหญ่ที่สุดของลาตินอเมริกา ที่แม้แต่ Amazon ยังตีไม่แตก

ตัวเลข MELI อ้างอิงงบ FY2025 (10-K) + earnings call Q1 FY2026 ผ่านบทวิเคราะห์ ผ่าธุรกิจ MELI บนเว็บนี้ — เป็นบันทึกการเรียนเพื่อการศึกษา ไม่ใช่คำแนะนำซื้อขาย

1. Network Economies: คุณค่าที่มาจาก "คนอื่น" ไม่ใช่จากสินค้า

นิยามตาม Helmer: ธุรกิจที่คุณค่าต่อผู้ใช้หนึ่งคนเพิ่มขึ้น เมื่อจำนวนผู้ใช้คนอื่นเพิ่มขึ้น — Benefit คือเมื่อเครือข่ายใหญ่ถึงจุดหนึ่ง ผู้เล่นอันดับหนึ่งให้คุณค่าที่เบอร์สองให้ไม่ได้ ต่อให้สินค้าเบอร์สองดีกว่า ส่วน Barrier คือผู้ท้าชิงไม่ได้แข่งเรื่องคุณภาพสินค้าอีกต่อไป แต่ต้องอุดช่องว่างของ "จำนวนคนอีกฝั่ง" ซึ่งเงินซื้อตรงๆ ไม่ได้

อย่าจับสับสนกับ Scale Economies: ตัวนั้นเป็นเรื่องฝั่งอุปทาน (ผลิตเยอะ → ต้นทุนต่อหน่วยต่ำ) ตัวนี้เป็นเรื่องฝั่งอุปสงค์ (คนใช้เยอะ → ของชิ้นเดิมมีค่ามากขึ้น) — ธุรกิจเดียวกันมีได้ทั้งคู่ แต่กลไกพังคนละแบบ และ Kill Conditions ก็คนละชุด

รูปแบบที่เจอบ่อยมี 3 ท่า:

  • เครือข่ายตรง — ผู้ใช้ทุกคนได้ประโยชน์จากผู้ใช้เพิ่ม (โทรศัพท์, LINE, WhatsApp)
  • เครือข่ายสองด้าน — สองกลุ่มดึงกันข้ามฝั่ง: ผู้ซื้อ↔ผู้ขาย (marketplace), ผู้ถือบัตร↔ร้านรับบัตร (เครือข่ายบัตร) — MELI อยู่ตรงนี้
  • เครือข่ายข้อมูล — ยิ่งใช้เยอะ ข้อมูลยิ่งทำให้สินค้าฉลาดขึ้นสำหรับทุกคน (โยงตอนนี้ท้ายเรื่อง: MELI ใช้ท่านี้ซ้อนอีกชั้นกับสินเชื่อ)

2. เดินเข้าจัตุรัสตลาด: 84 ล้านคนที่เรียกร้านค้ามาเอง

ภาพที่ถูกต้องของ MELI ไม่ใช่ "เว็บขายของ" แต่คือจัตุรัสตลาดของทั้งทวีป — ที่บราซิล เม็กซิโก อาร์เจนตินา รวมกันเป็นรายได้เกือบ 95% ของบริษัท ผู้ซื้อ active 84 ล้านคนเดินอยู่ในจัตุรัสนี้ทุกไตรมาส และนั่นคือแม่เหล็กเพียงตัวเดียวที่ร้านค้าต้องการ:

จุดที่ต้องสังเกตให้ดี: ในธุรกิจแบบนี้ สินค้า (ตัวเว็บ/แอป) ไม่ใช่ของที่ลูกค้าซื้อ — ลูกค้าซื้อ "คนอีกฝั่ง" ผู้ซื้อจ่ายเพื่อเจอร้านเยอะ ร้านจ่ายเพื่อเจอผู้ซื้อเยอะ คุณสมบัติของแอปเป็นแค่ทางผ่าน คัดลอกได้ในหกเดือน แต่ 84 ล้านคนคัดลอกไม่ได้

3. กำแพงไก่-ไข่: ทำไม Amazon เจาะไม่เข้า

นี่คือเหตุผลที่ Helmer ย้ำว่าสนาม network economies เป็นสนามแบบ winner-take-most และตัดสินกันเร็ว — การแข่งขันจริงเกิดตอนแย่งมวลวิกฤตช่วงแรก หลังจากนั้นเกมเกือบปิด ผู้ท้าชิงที่เหลือทำได้แค่สองทาง: หาสนามย่อยที่เครือข่ายเดิมยังอ่อน (Shopee/Temu เจาะกลุ่ม ultra-low-price จากจีน — ดู Bear Case ใน deep-dive MELI) หรือรอให้เทคโนโลยีเปลี่ยนสนามใหม่ทั้งกระดาน

4. เครือข่ายซ้อนเครือข่าย: Mercado Pago + Envíos

สิ่งที่ทำให้ MELI เป็นเคสสอนหนังสือคือมันไม่หยุดที่เครือข่ายเดียว — มันใช้ฐานผู้ใช้ marketplace เปิดเครือข่ายที่สองที่มีไก่-ไข่ของตัวเอง: Mercado Pago เริ่มจากกระเป๋าเงินในแอป วันนี้หลุดออกไปอยู่ตามหน้าร้านจริงทั้งทวีป

แต่จำจากตอนที่แล้วได้ไหม — ทุก Benefit ต้องถามหา Barrier และทุกอำนาจมีราคาของมัน: การเป็นเจ้าของเครือข่ายสินเชื่อแปลว่ารับความเสี่ยงเครดิตแบบธนาคารเข้ามาเต็มๆ — NIMAL (กำไรดอกเบี้ยหลังหักหนี้เสีย) ของ MELI ไหลลงจาก 36.2% (2023) → 17.8% (Q1'26) เพราะเร่งขยาย credit card และ NPL >90 วันแตะ 17.6% — เครือข่ายทำให้ปล่อยกู้ได้แม่นกว่าธนาคาร แต่ไม่ได้แปลว่าปล่อยเท่าไหร่ก็ได้ นี่คือเส้นแบ่งที่ต้องเฝ้าใน deep-dive MELI

5. หา Network Economies ในงบตรงไหน

เครือข่ายจริงทิ้งรอยไว้ในตัวเลขเสมอ — เช็คลิสต์ 4 ข้อ:

  • ทั้งสองฝั่งต้องโตพร้อมกัน — ผู้ซื้อโตแต่ผู้ขายหด (หรือกลับกัน) = วงล้อกำลังฝืด MELI: ผู้ซื้อ 84M โต ควบคู่ items sold, GMV และจำนวนผู้ขายที่ใช้ fulfillment (95.5% ของ volume)
  • Engagement ต่อหัวต้องไม่จาง — เครือข่ายแท้ยิ่งอยู่นานยิ่งใช้บ่อยขึ้น (ความถี่ซื้อ/ยอดซื้อต่อ cohort) ถ้าผู้ใช้ใหม่โตแต่คนเก่าเงียบลง นั่นคือการตลาด ไม่ใช่ network effect
  • Take rate เป็นเทอร์โมมิเตอร์อำนาจ — แพลตฟอร์มที่เครือข่ายแข็งขึ้นควรค่อยๆ เก็บส่วนแบ่งได้มากขึ้น (หรืออย่างน้อยคงที่) โดยผู้ขายไม่หนี — take rate ที่ต้องลดเพื่อรั้งผู้ขาย คือเครือข่ายที่กำลังเสียอำนาจ
  • ทดสอบ multi-homing — คำถามเดียวที่สำคัญสุด: ผู้ใช้เปิดสองแพลตฟอร์มพร้อมกัน ได้ง่ายแค่ไหน? ร้านค้าลงของทั้ง MELI และ Shopee ได้ (multi-home ง่าย = barrier บางลง) แต่ fulfillment + เงินค้างใน Pago + ประวัติเครดิต ย้ายยากกว่ามาก — MELI จึงถักสามชั้นนี้เข้าด้วยกัน

6. เมื่อเครือข่ายตาย: มันไม่ค่อยตายช้าๆ

ข่าวร้ายของอำนาจนี้: เพราะคุณค่าอยู่ที่ "คนอื่นอยู่ครบ" พอคนเริ่มย้าย มันจึง พังเป็นฝูง ไม่ใช่ทีละคน — MySpace ใหญ่กว่า Facebook มาก่อน แล้วหายไปในเวลาไม่กี่ปี เพราะ network effect ทำงานย้อนทางได้เท่าๆ กับทางขึ้น: เพื่อนเราย้าย → เหตุผลที่เราอยู่หายไป → เราย้าย → เหตุผลของคนถัดไปหายอีก

  • ผู้ชนะเฉพาะสนามย่อย — เครือข่ายระดับภูมิภาคแพ้เครือข่ายเฉพาะทางได้ (ของถูกจากจีน = สนามย่อยที่ MELI ไม่ได้แข็งที่สุด — Temu/Shopee เข้าตรงนี้)
  • เทคโนโลยีเปลี่ยนหน่วยของเครือข่าย — เครือข่ายเพื่อน (Facebook) โดนเครือข่ายความสนใจ (TikTok) ตีข้าง เพราะ TikTok ไม่ต้องง้อ social graph เลย — อย่าถามแค่ "ใครลอกเครือข่ายเราได้" ให้ถาม "ใครทำให้เครือข่ายเราไม่จำเป็น"
  • รัฐ/regulator หั่นเครือข่าย — เครือข่ายผูกขาดเชิญคำสั่ง interoperability (เปิดให้คู่แข่งต่อท่อ) ซึ่งเท่ากับยึด Barrier ไปแจก

7. Kill Conditions — สัญญาณว่าอำนาจนี้กำลังพัง

สำหรับหุ้นที่ thesis พึ่ง Network Economies:

  • ฝั่งใดฝั่งหนึ่งของเครือข่ายหดตัว ติดกัน 2-3 ไตรมาสทั้งที่อีกฝั่งยังโต → วงล้อเริ่มหมุนกลับ — อย่ารอให้หดทั้งสองฝั่ง ตอนนั้นสายเกินไป
  • Engagement ต่อหัวของ cohort เก่าลดลงต่อเนื่อง → เครือข่ายกำลังกลายเป็นแค่ traffic
  • Take rate ต้องลดเพื่อรั้งอีกฝั่ง (MELI: ลด >200bps/ปีต่อเนื่อง + 3P seller ย้ายออก ขณะคู่แข่งได้ fulfillment ครบบราซิล+เม็กซิโก) → Barrier ถูกเจาะแล้ว
  • เฉพาะ MELI: NIMAL ต่ำกว่า 15% พร้อม NPL >90 วันเกิน 20% → เครือข่ายข้อมูล-สินเชื่อที่เป็นชั้นในสุดกำลังซื้อ growth ด้วยความเสี่ยงที่ชดเชยไม่ไหว

สรุป: อำนาจที่ซื้อไม่ได้ด้วยเงิน — แต่พังได้ด้วยฝูงคน

Network Economies เป็นอำนาจที่ Barrier สูงที่สุดตัวหนึ่ง — เงินและความเก่งซื้อไม่ได้ ต้องอาศัยการมาก่อนและมวลวิกฤต — แต่ก็เป็นอำนาจที่ตายเร็วที่สุดเวลามันตาย เพราะคุณค่าทั้งหมดฝากไว้กับการที่ "ทุกคนยังอยู่" วินัยของคนถือหุ้นเครือข่ายจึงไม่ใช่เฝ้าราคา แต่เฝ้าชีพจรของทั้งสองฝั่งทุกไตรมาส

สองตอนแรกเราเจออำนาจที่ต้อง "สร้างมวล" — ใหญ่กว่า (scale) หรือคนเยอะกว่า (network) ตอนหน้ากลับหัวสิ้นเชิง: อำนาจของคนตัวเล็กที่ชนะเจ้าตลาดได้ด้วยการเลือกโมเดล ที่เจ้าตลาดเห็นแต่ไม่กล้าตาม เพราะตามเมื่อไหร่กำไรตัวเองพังเมื่อนั้น — ตอนที่ 3: Counter-Positioning ผ่านเรื่องที่ Netflix ฆ่า Blockbuster ทั้งที่ Blockbuster เคยมีโอกาสซื้อ Netflix ในราคา $50M แล้วหัวเราะใส่

อ่านเคสเต็มของ MercadoLibre ต่อได้ที่ ผ่าธุรกิจ MELI — โตเร็วที่สุด แต่ margin กำลังร่วง
