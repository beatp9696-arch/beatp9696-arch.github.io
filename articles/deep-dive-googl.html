ผ่าธุรกิจ GOOGL (Alphabet) — เครื่องโฆษณาที่ซ่อนคูเมือง "data" และเดิมพัน AI

คนใช้ Google ฟรีทุกวัน — แล้วบริษัทหาเงินมหาศาลได้ยังไง? คำตอบคือ มันขาย "ความตั้งใจซื้อ" ของคุณให้ผู้ลงโฆษณา ในเสี้ยววินาที และเบื้องหลังเครื่องพิมพ์เงินนี้คือคูเมือง data ที่สะสมมาสิบปี บทนี้อธิบายว่า Alphabet หาเงินยังไง คูเมืองอยู่ตรงไหน ขาที่สอง (Cloud/AI) ไปถึงไหน และจุดเปราะที่ใหญ่ที่สุดคืออะไร

ตัวเลขอ้างอิงงบที่เผยแพร่ (FY2025) และข้อมูลสาธารณะ ควรตรวจสอบกับ filing ล่าสุดก่อนใช้ตัดสินใจ เป็นการวิเคราะห์เพื่อการศึกษา ไม่ใช่คำแนะนำซื้อขาย — เน้นอธิบายโครงสร้างธุรกิจมากกว่ามูลค่า/ราคาเป้า

1. ธุรกิจนี้หาเงินจากอะไร

Alphabet คือบริษัทแม่ของ Google รายได้หลักกว่า ครึ่งหนึ่ง (55%+) มาจากโฆษณาบน Search (ยังไม่รวม YouTube และ Network) ลูกค้าที่จ่ายเงินจริงไม่ใช่ผู้ใช้ทั่วไป แต่เป็น ผู้ลงโฆษณา (advertiser) ที่อยากเข้าถึงคนตอนที่ ความตั้งใจซื้อสูงที่สุด — คือตอนที่คุณพิมพ์ค้นหา

หัวใจคือ โฆษณาบน Search มี ROI สูงผิดปกติ เพราะมันยิงตรงจังหวะ intent — คนที่ค้นว่า "ตั๋วเครื่องบินโตเกียว" กำลังจะซื้อตั๋วจริง ไม่ใช่แค่ดูเล่น advertiser จึงยอมประมูลแพง และ Google เก็บเงินทุกครั้งที่มีคนคลิก

2. ใครคือลูกค้า และทำไมเขาต้องใช้ (Reach)

เหตุผลที่ advertiser ต้องลงโฆษณากับ Google คือ ไม่มีที่ไหนเข้าถึงคนได้มากเท่า — Google มีผลิตภัณฑ์ที่มีผู้ใช้เกินพันล้านคนหลายตัว ทั้งหมดป้อน attention และ data กลับเข้าเครื่องโฆษณาเดียวกัน

นี่คือทั้ง reach (เข้าถึงคนได้กว้างสุด) และ distribution (เป็นประตูทางเข้าอินเทอร์เน็ตของคนทั้งโลก) ที่คู่แข่งไม่มี — และเป็นจุดที่ DOJ จับตา (ดู Bear Case)

3. Moat: คูเมืองอยู่ตรงไหน

คูเมืองที่แท้จริงของ Google ไม่ใช่ algorithm ที่ฉลาดกว่า แต่คือ data วงจรป้อนกลับ ที่สะสมมาสิบปีและคู่แข่งลอกไม่ได้ในเวลาสั้น:

นี่คือ Counter-Positioning แบบ Helmer — Google มี query และ click signal จากคนหลายพันล้านคนที่ใช้ train ranking และ ad model ได้แม่นกว่าใคร แม้คู่แข่งจะมี AI ที่เก่งกว่า ในบาง benchmark ก็ยังขาด data การใช้งานจริง ก้อนนี้

4. ขาที่สอง: Google Cloud + AI

ถ้า Search คือ cash cow ขาที่สองที่ตลาดให้ความสำคัญคือ Google Cloud — segment ที่โตเร็วที่สุด (ราว 10-12% ของรายได้) เพิ่งผ่านจุดคุ้มทุนและ margin กำลังขยาย

จุดที่ทำให้ Google ต่างคือมัน สร้างชิป AI ของตัวเอง (TPU) ไม่ต้องพึ่ง NVIDIA ทั้งหมด และมีโมเดล Gemini ของตัวเองครบ stack ตั้งแต่ชิป → โมเดล → cloud → ผลิตภัณฑ์ ปลายทาง (Search/Workspace) — integration แนวตั้งที่น้อยรายทำได้

Waymo ยังห่างไกล profitability และเป็น drag ต่อ margin แต่ถ้า commercial scale ได้จริงใน 3-5 เมืองใหญ่ มันคือ business model ใหม่ทั้งหมดที่ ยังไม่ถูก price-in ใน valuation ปัจจุบัน — เป็น optionality ไม่ใช่จานหลัก

6. ตัวเลขบอกอะไร

  • รายได้ระดับ $350B+ (FY2025 ประมาณการ) — Search ยังเป็น majority, Cloud ขยับเป็น 10-12% และโตเร็วสุด
  • Operating leverage สูง — Search/YouTube มี incremental margin สูงมาก, Cloud ผ่าน break-even แล้วและ margin กำลังขยาย, operating margin รวมราว 34%
  • Capital allocation เข้มขึ้น — buyback scale ใหญ่ขึ้นมากช่วง 2023-2025, เริ่มจ่าย dividend ครั้งแรกปี 2024, แต่ยังลงทุน AI infrastructure หนัก (CapEx พุ่ง)
ตัวเลขข้างบนเป็นข้อมูลสาธารณะ/training memory — ควรเปิดงบไตรมาสล่าสุดยืนยันก่อนใช้จริง บทนี้เน้น "เข้าใจว่าธุรกิจหาเงินยังไงและคูเมืองอยู่ตรงไหน" มากกว่าตัวเลขเป๊ะรายไตรมาส

7. Bull Case

  • Intent data เป็น Counter-Positioning ที่สะสมสิบปี — query/click signal จากคนพันล้านคน train ad model ได้ดีกว่าใคร replicate ไม่ได้ในระยะสั้น
  • Google Cloud margin expansion เพิ่งเริ่ม — ถ้า trajectory margin ขยับจาก low-teens ไปหา AWS/Azure (~25-30%) จะเพิ่ม earnings มากโดย revenue ยังโต
  • Waymo เป็น option value ที่ยัง underpriced — ถ้า robotaxi scale ได้จริง มูลค่าจะ unlock แบบไม่ได้ price in

8. Bear Case / จุดเปราะ

ทุก thesis ที่ดีต้องมองด้านมืดให้ชัด — สำหรับ Alphabet จุดเปราะมาจาก 2 ทางที่กระทบ cash cow โดยตรง:

และความเสี่ยงเชิงกฎหมายที่จ่อตัดกลไก defend share:

  • AI Search กำลัง test switching cost ของ Search — ถ้า informational query ย้ายไป AI assistant แม้ 10-15% โฆษณา Search ที่ fund ทั้ง Alphabet จะกระทบโดยตรง
  • DOJ antitrust คุกคาม distribution deal — ศาลตัดสินว่า Google ผูกขาด search; remedy ที่พิจารณารวมถึงห้ามจ่าย default search fee ให้ Apple/carrier ถ้า implement กระทบทันที
  • CapEx cycle ที่ aggressive อาจไม่ได้ return ตามคาด — ลงทุน AI หลักแสนล้าน ถ้า monetization ไม่ scale หรือถูก commoditize เร็ว ROIC จะน่าผิดหวังและกด valuation

9. Kill Conditions — อะไรจะทำให้ thesis พัง

วินัยสำคัญ: ต้องรู้ก่อนว่า "เหตุการณ์แบบไหนจะบอกว่าผมคิดผิด" สำหรับ GOOGL คือ:

  • Search query volume ลด YoY ครั้งแรก พร้อม Search revenue โตต่ำกว่า 5% → ถ้า volume หดและ pricing ช่วยไม่ทัน = AI assistant กำลัง structurally displace Search จริง ไม่ใช่แค่ narrative
  • DOJ remedy ห้ามจ่าย default search fee และศาลอุทธรณ์ยืน → distribution deal กับ Apple/carrier คือกลไก defend share ถ้าหายถาวร competitive dynamic เปลี่ยนทันที
  • Google Cloud โตต่ำกว่า 20% YoY ติดกัน 2 ไตรมาส พร้อม margin ไม่ขยาย → Cloud คือ thesis ที่สองที่รองรับ valuation ถ้าไม่ deliver Alphabet กลายเป็น mature ad company ที่ premium ไม่ justified

สรุป

GOOGL เป็น "wonderful business" ที่ moat ไม่ได้อยู่ที่ algorithm แต่อยู่ที่ intent data + reach + distribution ที่สะสมมาสิบปี บวกขาที่สอง (Cloud/AI ครบ stack ตั้งแต่ TPU) และ optionality (Waymo) จุดเปราะที่ใหญ่ที่สุดคือ AI search ที่อาจกิน cash cow และ DOJ ที่จ่อตัด distribution — ทั้งคู่มี Kill Condition ที่วัดได้ชัด ใครจะถือต้องตอบให้ได้ว่า ถ้า search volume เริ่มหดหรือ DOJ สั่งจริง จะทำยังไง
